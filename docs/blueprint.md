# **App Name**: NoteWise

## Core Features:

- Smart Summaries: AI-powered summarization tool to create concise summaries of longer notes.
- Contextual Suggestions: Smart suggestions tool to suggest related topics, keywords, and additional information based on note content.
- Rich Note Editor: Intuitive note editor with rich text formatting and multimedia embedding capabilities.
- Tag-Based Organization: Categorize notes with tags and a powerful search function to find the right notes, quickly.
- Export Notes: Export notes as text and/or Markdown.

## Style Guidelines:

- Primary color: Calm light-blue (#64B5F6) to promote focus and clarity.
- Background color: Light gray (#F0F4F8), creating a soft, distraction-free canvas.
- Accent color: Muted purple (#9575CD) for highlighting key actions and elements.
- Body and headline font: 'Inter' (sans-serif) for clear, readable text.
- Use minimalist, consistent icons to represent note categories and actions.
- Clean and organized layout for easy navigation.
